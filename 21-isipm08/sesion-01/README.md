## PROTOCOLOS!
+ MQTT (Message Queuing Telemetry Transport): Lenguaje estandar para el internet de las cosas, protocolo de transporte de mensajería de manera eficiente, basado en el modelo cliente - servidor utiliza broker.
+ HTTPS (HyperText Transfer Protocol Secure): Protocolo que permite transferencia de datos entre una navegador web y un servidor, permitiendo que la información sea privada e íntegra.
+ OSC (Open Sound Control): Protocolo de comunicación diseñado para el intercambio de comunicación entre dispositivos multimedia, destacando su velocidad y precisión.

## CONCEPTOS
+ SEMIÓTICA: Disciplina científica que estudia los signos, símbolos y sistemas, su funcionamiento y como se producen para transmitir e interpretar mensajes en la comunicación humana.
+ IoT: Red de objetos físicos, equipados con sensores, software y tecnologías, permitiendo el intercambio de datos a través de internet sin la necesidad de la intervewnción humana.
+ PROCESSING: Entorno de desarrollo integrado (IDE), lenguaje programación de código abierto basado en Java (creación 2001 Casey Reas - Ben Fry MIT media lab). Usado para crear arte generativo, animaciones, instalaciones interactivas y visualización de datos, con la idea un tipo de sketchbook, de que se pudiera realizar códigos y programación fuera tan fácil como dibujar en papel
+ BROKER: Componente que actúa como intermediario para poder gestionar la comunicación entre diferentes aplicaciones y servicios. "El que envía la información no necesita saber quién la recibe ni dónde está, solo se la entrega al Broker".
+ P5.JS: Biblioteca de código abierto para el aprendizaje creativo de la programación. Permite crear gráficos, animaciones, visualizaciones de datos e interacciones complejas usando código.
